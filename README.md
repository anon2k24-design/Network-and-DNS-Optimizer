# Network and DNS Optimizer

A Windows network tuning script focused on lowering latency and improving connection responsiveness for gaming.

## Features

- Flushes DNS cache.
- Resets TCP/IP settings.
- Applies latency-focused TCP tweaks.
- Updates DNS servers.
- Optimizes network adapter settings.

## How to use

Run PowerShell as Administrator, then:

```powershell
.\network-dns-optimizer.ps1
```

## Best for

- Competitive gaming.
- High-ping troubleshooting.
- Users who want a quick network refresh.
